Java基础

Posted on 2019-11-11 | In Java

创建、编译、运行Java程序

  • 命令行编译Java程序：

    注意：Java区分大小写

    javac Welcome.java

    java Welcome

  • 源代码的文件名必须与公共类的名字相同，并用.java作为扩展名

  • 在一个源文件中，只能有一个公有类，但可以有任意数目的非公有类

  • 每个Java应用程序都必须有一个main方法

数据类型

  • Java没有任何无符号类型

整型

类型 存储要求 备注
int 4字节
long 8字节 4000000000L
short 2字节
byte 1字节 0b1001

浮点类型

  • Java中默认浮点类型为double类型

  • 浮点数不适用于禁止出现四舍五入的计算中，如果需要在数值计算中不含有任何舍入误差，应该使用BigDecimal类

类型 存储要求 备注
float 4字节 3.14F，没有后缀F默认为double类型
double 8字节

常量

  • 利用关键字final指示常量，关键词final表示这个变量只能被赋值一次，常量名使用全大写
  • 关键字static final设置一个常量，使得这个常量可以在一个类的多个方法中使用，成为类常量
  • 如果一个常量被声明为public，那么其他类的方法也可以使用这个常量

检测字符串是否相等

  • 可以用equals方法检测两个字符串是否相等，相等返回true，不等返回false

    s.equals(t);

  • 可以用equalsIgnoreCase方法检测两个字符串是否相等，而不区分大小写

    “Hello”.equalsIgnoreCase(“hello”);

  • 一定不能使用==运算符检测两个字符串是否相等。这个运算符只能确定两个字符串是否在同一个位置上。

空串与Null串

  • 空串是长度为0的字符串，合法初始化。

    空串是一个Java对象，有自己的串长度和内容

    可以调用 if(str.length()==0) 或 if(str.equals(“”)) 来检测字符串是否为空

  • null表示目前没有任何对象与该变量相关联，变量没有被初始化

    if(str == null)

  • 检查一个字符串既不是null也不为空串，首先要检查str不为null，因为如果在一个null上调用方法会出错

    if(str != null && str.length() != 0)

创建数组

  • 创建一个数字数组时，所有元素都初始化为0。boolean数组的元素会初始化为false。对象数组的元素则初始化为一个特殊值null
  • array.length可以获得数组中的元素个数
  • 一旦创建了数组，就不能在改变它的大小
