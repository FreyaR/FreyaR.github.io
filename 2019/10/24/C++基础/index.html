输入

cin使用空白（空格、制表符和换行符）来确定字符串的结束位置，这意味着cin在获取字符数组输入时只读取一个单词。

面向行的输入

getline()

getline()读取整行，它使用通过回车键输入的换行符来确定输入结尾。

getline()参数：第一个参数用来存储输入行的数组的名称。第二个参数是要读取的字符数，如果这个参数为20，则函数最多读取19个字符，余下的空间用于存储自动在结尾处添加的空字符。

cin.getline(name,20)

cin.getline(name1,20).getline(name2,20)

读取一行到string对象中：getline(cin,str);

get()

get()不再读取并丢弃换行符，而是将其留在输入队列中

使用不带任何参数的cin.get()调用可读取下一个字符（即使是换行符）

处理数据

整型

整型short、int、long、long long

  • short：至少为16位
  • int：至少与short一样长
  • long：至少为32位，且至少与int一样长
  • long long：至少为64位，且至少与long一样长

在老式的PC中，int的宽度为16位。而在一些新的PC中，int的宽度为32位

字符

  • 获取字符的编码

  • C++ 新增类型：char16_t和char32_t

    char16_t：无符号16位，使用前缀u表示char16_t字符常量和字符串常量，u’C’

    char32_t：无符号32位，使用前缀U表示char32_t字符常量和字符串常量，U’R’

const限定符

常量被初始化后，其值就被固定了，编译器不允许再修改该常量的值

浮点数

  • float：至少为32位，通常位32位，1.23f
  • double：至少为48位，且不少于float，通常为64位，2.345324E28
  • long double：至少和double一样多，为80，96或128位，2.2L

数组

数组的初始化规则

模板类vector

动态数组，可以在运行阶段设置vector对象的长度，可在末尾附件新数据，可在中间插入新数据，使用new和delete来管理内存

模板类array

长度固定，使用栈，比数组更方便更安全

数组、vector、array对象的不同

  • 无论是数组、vector还是array，都可以使用标准数组表示法来访问各个元素
  • array和数组存储在栈中，vector存储在堆中
  • 可以将一个array对象赋给另一个array对象，而对于数组必须逐元素复制数据
  • vector和array可以使用成员函数at()。使用at时可以在运行期间捕获非法索引。例：vi.at(1)

结构体

可以将结构作为参数传递给函数，也可以让函数返回一个结构。另外，还可以使用赋值运算符（=）将结构赋给另一个同类型的结构

结构数组

函数

C++对于返回值的类型有一定限制，不能是数组，但可以是其他任何类型——整数、浮点数、指针，甚至可以是结构和对象

函数与数组

在C++中，当且仅当用于函数头或函数原型中，int *arr和int arr[]的含义相同

传递常规变量时，函数将使用改变量拷贝。传递数组时，函数将使用原来的数组。传递数组时，并没有将数组内容传递给函数，而是将数组的位置（地址）提交给函数

为将数组类型和元素数量告诉数组处理函数，请通过两个不同的参数来传递它们：void fillArray(int arr[]，int size)

用const保护数组

使用普通参数时，这种保护将自动实现，这是由于C++按值传递，并且函数使用数据的副本。然而接受数组名的函数将使用原始数据。为防止函数修改数组内容，可在声明形参时使用关键字const

void show_array(const double ar[],int n);

const和指针

  • 让指针指向一个常量对象，这样可以防止使用该指针来修改所指向的值

  • 将指针本身声明为常量，这样可以防止改变指针指向的位置

    尽可能将指针参数声明为指向常量数据的指针：

    • 这样可以避免由于无意间修改数据而导致的编程错误
    • 使用const使得函数能够处理const和非const实参，否则只能接受非const数据
